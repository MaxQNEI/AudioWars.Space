# Install

1. npm i
2. cd public/
3. ln -s ../dist/index.js .
4. ln -s ../dist/assets/ .
5. npm run build

* OR Oneline: `npm i; cd public/; ln -sf ../dist/index.js .; ln -sf ../dist/assets/ .; npm run build`
* OR Bash script: `./install.sh`
